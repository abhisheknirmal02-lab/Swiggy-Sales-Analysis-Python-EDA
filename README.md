📊 Swiggy Sales Analysis — Python EDA

This project focuses on analyzing Swiggy sales data using Python to uncover trends, patterns, and business insights that can help in better decision-making.

The goal was to move beyond raw data and understand how revenue behaves across time, geography, and customer preferences.

🔍 Project Overview

Using Python-based data analysis and visualization, this project explores:

Monthly revenue trends to identify growth patterns
Daily sales distribution across weekdays
Customer preference split between veg and non-veg orders
State-wise revenue contribution
Top-performing cities by sales

The analysis helps in understanding demand patterns, peak periods, and regional performance.

📌 Key Insights
Revenue shows fluctuations month-to-month with clear recovery trends
Weekend sales are slightly higher compared to weekdays
Veg orders contribute a larger share of total revenue
Certain states dominate overall sales performance
A few major cities drive a significant portion of revenue

🛠️ Tools & Technologies Used
Python
Pandas & NumPy (Data Cleaning & Analysis)
Matplotlib & Seaborn (Visualization)
Jupyter Notebook
